# 💻 GS에너지 PC 자산관리 시스템

GS에너지 · GS파워 · 인천종합에너지 3개 사업장의 PC 자산을 통합 관리하는 웹 시스템입니다.

---

## 개발 환경

| 항목 | 내용 |
|------|------|
| Language | Python 3.14 |
| Framework | Flask (vendor 폴더에 내장) |
| Database | SQLite (`assets.db`) |
| Frontend | HTML / CSS / JavaScript (단일 파일 SPA) |
| OS | Windows |

---

## 실행 방법

```bash
# 1. 환경변수 파일 생성 (.env)
#    SECRET_KEY=랜덤문자열 을 .env 파일에 저장

# 2. 서버 실행
python app.py

# 3. 브라우저 접속
http://localhost:5000
```

> `.env` 파일은 보안상 Git에 포함되지 않습니다.  
> 최초 실행 전 `.env` 파일을 직접 생성해야 합니다.

---

## 주요 기능

- **대시보드** — 전체 자산 현황 KPI, 사업장·상태·기기종류별 통계, 교체 검토 기준 설정
- **자산 목록** — 검색·필터·컬럼 정렬, 인라인 편집, 상세 보기
- **자산 등록** — 필수 항목 유효성 검사, S/N 중복 방지
- **엑셀 업로드** — `.xlsx` 일괄 등록, 중복 S/N 덮어쓰기 옵션
- **로그인 / 권한 관리** — 관리자·일반 역할 구분, 세션 기반 인증
